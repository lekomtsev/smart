    Сайт будет сделан в виде SPA на Vue.js
    
#### Структура проекта
    -- на данный момент еще в разработке
        - будет основная часть сайта

#### Project setup
```
npm install
```

#### Compiles and hot-reloads for development
```
npm run serve
```

#### Compiles and minifies for production
```
npm run build
```

#### Lints and fixes files
